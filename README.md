# HeVisual Assets

Binary asset repo for [HeVisual](https://github.com/codyhex/HeVisual). Mounted as a git submodule at `public/_assets/`.

## Layout

- `exhibitions/<slug>/` — per-exhibition images (hero, gallery, material reference)
- `textures/` — shared site textures (silk, paper, etc.)
- `reference/` — design source / mood material

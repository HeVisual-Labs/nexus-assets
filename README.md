# HeVisual Assets

Source repo: [HeVisual-Labs/nexus-assets](https://github.com/HeVisual-Labs/nexus-assets). In the [nexus-web](https://github.com/HeVisual-Labs/nexus-web) app it is checked out as the git submodule at `public/_assets/`.

## Layout

- `exhibitions/<slug>/` — per-exhibition images (hero, gallery, material reference)
- `textures/` — shared site textures (silk, paper, etc.)
- `reference/` — design source / mood material
